# Parallel-Practice

A collection of parallel computing practice implementations, focusing on GPU-accelerated matrix multiplication using CUDA.

## Overview

This repository contains implementations and experiments with SGEMM (Single-precision General Matrix Multiply) operations using NVIDIA CUDA for parallel computing on GPUs.

## Contents

- **`sgemm/`** - SGEMM implementation with optimization techniques
  - `sgemm.cu` - CUDA kernel implementations for matrix multiplication
  - `README.md` - Detailed documentation for this implementation

- **`sgemm-1/`** - Alternative SGEMM implementation
  - `sgemm.cu` - CUDA kernel implementations for matrix multiplication
  - `README.md` - Detailed documentation for this implementation

## Requirements

- NVIDIA GPU with CUDA support
- CUDA Toolkit
- C/C++ compiler compatible with your CUDA version

## Purpose

This repository serves as a learning resource and practice ground for:
- Parallel programming concepts
- GPU computing with CUDA
- Matrix multiplication optimization techniques
- Performance tuning and benchmarking

## Getting Started

Navigate to each subdirectory for specific implementation details and build instructions. Each directory contains its own README with more information about that particular implementation.
